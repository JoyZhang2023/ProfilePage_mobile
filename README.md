# ProfilePage_mobile
AD340 Assignment

## Object
Create a simple Expo application that demonstrates the use of React Native Core Components including View, Text, Image, ScrollView, and TextInput. Your goal is to construct your profile page that showcases these components' functionality and interactivity.

## Command
use npx-expo-start to start the project
